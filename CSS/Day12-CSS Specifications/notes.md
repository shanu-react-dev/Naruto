## CSS Specificity

_There are various ways to add CSS to our HTML Elements and tags and each ways having there own Priority. Among Inline, Internal and External Inline CSS will have the highest priority means if a tag is having style with all these three ways then inline only will be applicable._

_Internal and External CSS will work on cascading means which one is used latest that will work._

## @import

_This is also a way to add the CSS. By this method we can able to import any other CSS file or any libraries or css framework. It should be used in CSS file using syntax `@import url("filename.css")`._

## History of CSS

_CSS (Cascading Style Sheets) was introduced in 1996 by Haekon Wium Lee. Currently we are using CSS3. It is the latest version of CSS._
